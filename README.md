# The-Archer-s-Gambit
In this assignment, you are to develop a Heuristic Minimax Agent with Alpha-Beta Pruning for a custom strategy game called Archer’s Gambit. Unlike simpler games like Tic-Tac-Toe, Archer’s Gambit features a high branching factor (b \approx 60–80) and a dynamic board that shrinks as the game progresses.
